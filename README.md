# 🍕 Food App — Java Desktop/Web Application

A Java-based food application project built using IntelliJ IDEA. This project serves as a foundation for a food ordering or food management system built with Java.

---

## 📌 Description

FoodApp is a Java application project structured using IntelliJ IDEA's module system. The project is set up with a standard Java module configuration and serves as a starting point for building a food-related application — whether for ordering, menu management, or food delivery tracking. The clean project structure makes it easy to extend with additional features like database integration, REST APIs, or a front-end UI.

## ⚙️ Setup Instructions

### Prerequisites
- Java 8+
- IntelliJ IDEA (Community or Ultimate)

### 1. Clone the Repository
```bash
git clone https://github.com/priya-zha/foodapp.git
cd foodapp
```

### 2. Open in IntelliJ IDEA
- Launch **IntelliJ IDEA**
- Select **File → Open** and navigate to the cloned project folder
- IntelliJ will auto-detect the module configuration from `foodapp.iml`

### 3. Build and Run
- Use IntelliJ's **Build → Build Project** to compile
- Run the main class using the **Run ▶** button
